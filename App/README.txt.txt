SETTING UP DEPENDENCIES AND ENVIRONMENT
1. Install Python 3.12.4, with pip 24.1.2
2. Install SWI-Prolog and add swipl/bin to environment variables path
3. Install Visual Studio Build for C/C++ (approximately 5-6GB)
4. Install other packages by running this in the folder command line: 
	pip install Pillow janus-swi
5. Create a new user variable in system properties
	Variable Name: SWI_HOME_DIR
	Variable Value: C:\Program Files\swipl
	**Note: the variable value may change depending on your SWI-Prolog installation settings


RUNNING THE APP
1. Open the App Folder and double click app.exe

~ Enjoy! ~
